# Upstox Automated Login Launcher

Automates the Upstox web login process and directly opens the trading terminal.

## What it does
- User selects the account holder name
- User enters the mobile number once
- The program automatically:
  - Generates and enters TOTP
  - Enters PIN
  - Handles page navigation
- Lands the user on the Upstox trading terminal without further interaction

## Purpose
- Reduce repetitive manual login steps
- Enable faster, consistent access to the trading platform
- Support frequent daily usage

## Scope
- Local execution only
- Designed for personal workflow automation
- No trade execution or account modification
